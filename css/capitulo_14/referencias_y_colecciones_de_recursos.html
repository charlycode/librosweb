14.4. Referencias y colecciones de recursos

  • Especificación oficial de CSS 2.1: la que incluyen todos los navegadores actuales.
  • Especificación oficial de CSS 3: la versión que está reemplazando a la anterior en todos los navegadores modernos.
  • delicious.com/webDesign: colección de recursos relacionados con todos los aspectos del diseño web. Lista creada y mantenida por el diseñador Miguel Sánchez.

  • Foros del Web: incluyen foros específicos dedicados a resolver dudas relacionadas con el diseño web con CSS.
