unicode-bidi

DefiniciónModifica el comportamiento del algoritmo utilizado para el texto Unicode bidireccional
Valores
permitidos		Alguno o todos los siguientes valores y en cualquier orden:
  • normal
  • embed
  • bidi-override
  • inherit
Valor inicialnormal
Se aplica aTodos los elementos
Válida enmedios visuales
Se heredano
